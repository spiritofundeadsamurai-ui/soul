What Makes Soul Different

Soul isn't just another chatbot. It's your personal AI that lives on your machine, remembers everything, and grows with you.

🧠

Persistent Memory

Everything Soul learns is stored locally in SQLite. Conversations, knowledge, patterns, wisdom — nothing is forgotten.

🔒

Privacy First

Your data never leaves your machine. Works fully offline with Ollama. No cloud dependency, no data collection.

⚡

308 Real Tools

Not just chat — Soul can research, code, manage goals, track time, create documents, analyze emotions, and more.

🎓

Grows Smarter

Learning paths, pattern extraction, feedback loops, and spaced repetition. Soul learns from every interaction.

👥

Soul Family

Spawn specialized child agents. A researcher, a coder, a writer — they collaborate and share knowledge.

🌐

Any LLM Brain

Get Started in 3 Steps

From zero to your own AI companion in under 5 minutes.

1

Install

npm install -g soul-ai

2

Setup

soul-setup
Choose your brain, set passphrase

3

Chat

soul
Start talking. Soul remembers.

# Install Soul globally
$ npm install -g soul-ai

# Run the setup wizard
$ soul-setup
Choose brain: [1] Ollama (free, local) [2] Cloud API
Pulling qwen3:14b... Done!
Master passphrase: ****

# Start chatting
$ soul
Soul AI Agent — Ready
You > Hello, who are you?
Soul > I'm Soul, your personal AI companion...

# Or use as MCP server in Claude Code
$ claude mcp add soul -- npx soul-mcp

# Or run the web dashboard
$ soul-server
Web UI: http://localhost:47779/
